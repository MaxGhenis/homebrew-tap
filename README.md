# Homebrew Tap

Custom Homebrew tap for MaxGhenis apps.

## Installation

```bash
brew tap maxghenis/tap
```

## Available Casks

### RAMBar

Retro-styled menu bar RAM monitor for macOS.

```bash
brew install --cask rambar
```

Features:
- Real-time memory usage in menu bar
- Expandable Claude Code sessions and Chrome tabs
- Click-to-activate for apps (Python, VS Code, Slack, etc.)
- Memory pressure warnings
- Retro cyberpunk UI theme
